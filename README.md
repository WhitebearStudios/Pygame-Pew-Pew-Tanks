# Pygame-Pew-Pew-Tanks

This is the predecessor to the published Pew Pew Tanks made with Unity.

This was my first time building my own game systems from the ground up like gravity, collisions, procedural terrain, and computer-controlled enemies.
I coded my own game object hierarchy, adding new mechanics in each subsequent child class, like gravity.

After installing Python, Pygame and Tkinter, you should be able to play the game by running main.py.
